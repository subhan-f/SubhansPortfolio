# Tailwind CSS v4: What's New and How to Migrate

Tailwind CSS v4 is a ground-up rewrite that brings significant performance improvements and a modernized developer experience. Here's everything you need to know.

## Key Changes

### CSS-First Configuration

The biggest change is moving configuration from `tailwind.config.js` to CSS:

```css
@import "tailwindcss";

@theme {
  --color-primary: #1cd8d2;
  --color-secondary: #302b63;
  --font-family-heading: 'Poppins', sans-serif;
}
```

No more JavaScript config files — everything lives in your stylesheet.

### Lightning CSS Engine

Tailwind v4 is built on **Oxide**, a new high-performance engine written in Rust. The results speak for themselves:

- **10x faster** full builds
- **100x faster** incremental builds
- Automatic vendor prefixing (no more autoprefixer)

### Native Cascade Layers

Tailwind v4 uses CSS `@layer` natively:

```css
@layer base {
  /* Base styles */
}

@layer components {
  /* Component styles */
}

@layer utilities {
  /* Utility overrides */
}
```

## Migration Guide

### Step 1: Update Dependencies

```bash
npm install tailwindcss@latest @tailwindcss/vite
```

### Step 2: Replace Plugin with Vite Integration

```ts
// vite.config.ts
import tailwindcss from '@tailwindcss/vite';

export default {
  plugins: [tailwindcss()],
};
```

### Step 3: Update Your CSS

Replace the old directives:

```css
/* Before */
@tailwind base;
@tailwind components;
@tailwind utilities;

/* After */
@import "tailwindcss";
```

### Step 4: Move Theme Config to CSS

Migrate your `tailwind.config.js` theme values to `@theme` blocks in CSS.

## New Utility Classes

Tailwind v4 introduces several new utilities:

- **`text-wrap-balance`** — For balanced text wrapping
- **`size-*`** — Shorthand for `width` + `height`
- **Container queries** — Built-in `@container` support
- **`field-sizing-content`** — Auto-sizing textareas

## Conclusion

Tailwind CSS v4 is faster, simpler, and more powerful. The migration path is straightforward, and the developer experience improvements alone make it worth the upgrade.
